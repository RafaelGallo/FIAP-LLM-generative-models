## Introduction

The rise of **Large Language Models (LLMs)** is transforming the way we build Artificial Intelligence applications. Models such as **GPT**, **Gemini**, and **LLaMA** are capable of understanding and generating text with high accuracy, opening new possibilities for automation, data analysis, and the creation of intelligent systems. This repository brings together all the hands-on material from the **Generative Models with LLMs** course at **FIAP — Data Science & Artificial Intelligence (DSeAI)**, covering everything from core
concepts to advanced applications using modern frameworks and state-of-the-art tools.

## About the Course

The course progressively covers the main pillars of LLM-based development. It starts with an introduction to generative models and their underlying architectures, then evolves into advanced techniques such as **Multimodal Models**, which combine text and image understanding, and **LangChain**, a widely adopted framework for orchestrating LLM pipelines.

Next, the course explores **VectorDB** — vector databases such as **FAISS** and **Chroma** which are essential for semantic storage and retrieval of information. Finally, it dives deep into **RAG (Retrieval-Augmented Generation)**, a technique that combines information retrieval with text generation, enabling models to respond based on external and up-to-date data sources. All content is delivered through hands-on **Jupyter Notebooks**, with real-world examples and exercises that reinforce learning at every stage of the course.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/RafaelGallo/FIAP-LLM-generative-models.git
   ```

2. Navigate to the project folder:
   ```bash
   cd FIAP-LLM-generative-models
   ```

3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Launch the notebooks with Jupyter:
   ```bash
   jupyter notebook
   ```
